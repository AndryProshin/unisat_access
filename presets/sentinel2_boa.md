# Пресет sentinel2_boa.json

## Описание данных

Пресет предназначен для работы с данными спутников Sentinel-2 (уровень обработки L2A) — атмосферно скорректированные значения яркости на уровне поверхности (BOA — Bottom Of Atmosphere).

### Спутники
- SENTINEL-2A
- SENTINEL-2B
- SENTINEL-2C

### Прибор
MSI_BOA — MultiSpectral Instrument (атмосферная коррекция)

### Станция приёма
ESA — Европейское космическое агентство
